# External Data Workspace

This directory is intentionally ignored except for this README.

Raw public datasets must be downloaded locally and kept immutable under:

- `data/raw/external/kaggle-lsm-alphabet`
- `data/raw/external/mendeley-mosl-v1`

Do not commit images, videos, archives, extracted landmarks, credentials, or large model artifacts.
Use the Makefile targets documented in the project README to download metadata, import local archives,
audit structures, and build manifests.
