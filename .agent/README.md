# Agent Workspace Notes

`.agent/` contains repository-level continuity instructions for Codex-style maintenance
work in this workspace.

Committed content may include:

- `CONTINUITY.md` entries that summarize plans, decisions, progress, discoveries, and
  outcomes needed by future maintainers;
- repository-level collaboration instructions that are useful beyond one local machine.

Do not commit credentials, local shell history, generated logs, cache files, personal
configuration, absolute machine-specific paths, or temporary agent output.
