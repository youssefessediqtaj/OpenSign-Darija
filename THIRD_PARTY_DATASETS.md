# Third Party Datasets

## Mendeley MoSL v1

`mendeley_mosl_v1` is registered as the data source for isolated Moroccan Sign Language word videos.

- Title: A Dataset for Moroccan Sign Language (MoSL)
- DOI: 10.17632/23phgyt3mt.1
- License: CC BY 4.0
- Attribution: `docs/attributions/mendeley-mosl-v1.md`
- Related article: DOI 10.1016/j.dib.2025.112395

The ScienceDirect article is a documentation source for the Mendeley dataset and must not be counted as a separate dataset.

## Kaggle Moroccan LSM Alphabet

`kaggle_moroccan_lsm_alphabet` is registered for `ALPHABET_STATIC`, but its license is `TO_VERIFY`.
It is disabled for training until official Kaggle metadata confirms a compatible license.
