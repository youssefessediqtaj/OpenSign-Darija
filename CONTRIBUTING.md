# Contributing

Merci de contribuer a OpenSign Darija.

## Principes

- Respecter la separation frontend, API et inference.
- Ne pas ajouter de collecte video sans consentement explicite et documentation.
- Ne pas presenter le prototype comme certifie.
- Ajouter ou mettre a jour les tests avec chaque changement significatif.
- Garder les contrats API compatibles avec le frontend.

## Qualite

```bash
make lint
make test
```

Ouvrez une pull request avec un resume clair, les risques et les commandes de verification executees.
