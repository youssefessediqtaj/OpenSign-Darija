# Kaggle Alphabet Dataset

Registered source: `kaggle_moroccan_lsm_alphabet`.

The dataset is not enabled for training until official Kaggle metadata confirms a compatible license.
Use `make dataset-download-kaggle-alphabet` with local Kaggle credentials to fetch metadata safely.

The alphabet pipeline lives in `ml/datasets/alphabet/` and produces manifests with original labels,
normalized labels, canonical letter codes when known, review status, checksums, and relative paths.
