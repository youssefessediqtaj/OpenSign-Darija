# External Datasets

OpenSign Darija separates public external sources by task:

- `ALPHABET_STATIC`: static alphabet images from Kaggle, disabled until license verification.
- `WORD_ISOLATED`: isolated word/sign videos from Mendeley MoSL v1.

ScienceDirect DOI `10.1016/j.dib.2025.112395` documents the Mendeley dataset and is not a separate data source.

Raw archives, extracted files, videos, images, landmarks, credentials, and large models stay outside Git.
