# Darija Latin Conventions

La sortie latine standard seedee evite de melanger les conventions:

- `بغيت -> bghit`
- `الما -> lma`
- `فين -> fin`
- `طبيب -> tbib`
- `عاونوني -> awnouni`

Arabizi est reserve comme variante distincte (`LATIN_ARABIZI`) et ne doit pas etre melange avec la sortie standard.
