# External Dataset Licenses

Allowed license identifiers are configurable with `EXTERNAL_DATA_ALLOWED_LICENSES`.

Default allow-list:

- `CC0-1.0`
- `CC-BY-4.0`
- `CC-BY-SA-4.0`
- `ODC-BY-1.0`

This allow-list is an engineering gate, not legal advice. Kaggle sources remain disabled when metadata is missing or incompatible.
