# Manual Message Testing

Checklist:

1. ouvrir `http://localhost:8081/app/messages/new`;
2. ajouter un mot manuel;
3. generer;
4. verifier Darija arabe RTL;
5. modifier l'arabe;
6. verifier que le latin ne change pas seul;
7. finaliser;
8. ouvrir l'historique;
9. mettre en favori;
10. dupliquer;
11. supprimer la copie.

Depuis reconnaissance:

1. ouvrir `/app/recognition`;
2. capturer;
3. confirmer;
4. cliquer `Ajouter au message`;
5. verifier l'item ajoute.
