# Future Text-To-Sign Research

The local source project includes a SignLLM-inspired Arabic/French text-to-sign pose generation pipeline. It is not part of the current OpenSigne Darija recognition milestone.

Kept as future research only:

- Text to generated sign pose.
- AraBERT or other text encoders.
- Prompt2LangGloss.
- Transformer pose generators.
- Reinforcement-learning pose loss.
- Priority Learning Channel.
- Virtual avatar or generated skeletal sign video.
- BLEU/ROUGE evaluation for generated sign sequences.

These components are not active runtime dependencies and were not copied into production services.
