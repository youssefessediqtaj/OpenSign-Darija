# Semantic Concepts

Les concepts semantiques separent le signe reconnu de l'intention linguistique.

Tables principales:

- `semantic_concepts`
- `sign_semantic_mappings`
- `linguistic_entries`
- `message_templates`

Concepts de demonstration:

- `AFFIRMATION_YES`
- `NEGATION_NO`
- `REQUEST_HELP`
- `OBJECT_WATER`
- `PERSON_DOCTOR`
- `HEALTH_PAIN`
- `POLITENESS_THANKS`
- `ACTION_WANT`
- `QUESTION_WHERE`
- `EMERGENCY_ALERT`

Ces mappings sont des donnees de demonstration et demandent validation communautaire.
